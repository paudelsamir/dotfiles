My Hyprland setup based on end-4/dots-hyprland and quickshell with some productive tweaks.

## Key Stuff:
- GPU monitoring in status bar
- Network speeds display  
- Pomodoro /timer
- Todo system that persists
- Monthly goals tracker
- Better keybinds 
- Notifying widgets
- Minimal Quickshell setup..

## Key Keybindings
- `Super + Q` - Close window
- `Super + E` - File manager (Dolphin)
- `Super + R` - VS Code
- `Super + V` - Toggle clipboard(gradia)
- `Super + Space` - App launcher
- `Super + Tab` - Overview/window switcher
- `Super + Shift + S` - Screenshot
- `Super + Shift + R` - Screen recording toggle
- `Super + tab` - Switch to next workspace
- `Super + Shift + tab` - Switch to previous workspace
- `Alt + [1-0]` - Move windows to workspace
- and many more.....
